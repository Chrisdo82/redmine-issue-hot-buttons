Issue Hot Buttons by mikekolganov

Plugin for Redmine that add some often used actions to issue page.

Install

  • Download plugin
  • Extract to vendors/plugins of installed Redmine
  • Restart web server

Author

Thumbtack Inc., Mike Kolganov (mike.kolganov@gmail.com)
